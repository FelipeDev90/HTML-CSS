Texto marcado

Podemos criar um texto marcado usando a tag MARK

Podemos trocar a cor do MARK usando um STYLE de CSS(mark style = "background-color: coloque a cor aqui;")

Textos grandes e pequenos

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
